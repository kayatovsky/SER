model - переобучение модели
work - файл для реальных тестов с вашей речью
exp - файл для экспериментов над нейросетью
totaldata - датасет из 3 датасетов - RAVDESS, TESS, без названия

В папке result уже есть обученные модели, чтобы протестировать свою речь не нужно заново обучать их.
В папке graphics содержатся графики вашей речи после запуска work.
